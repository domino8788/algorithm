```
BFS를 이용해 풀어야하는 문제
그 외 방안도 물론 있으나
성공 케이스는 장애물과 유령을 고려해 목적지에 도착할 경로를 찾는다면 정답을 낼 수 있으나
실패 케이스가 발생하고서 복귀 후 재시도를 하기엔 로직이 너무 복잡하고 반례에 취약하다.

BFS를 이용하면 남우가 시작점에서 도착점까지 가는 비용을 유령, 장애물을 고려해 계산할 수 있다.
다음 방향에 장애물이 있다면 진입을 넘기고
다음 방향의 비용보다 유령의 비용이 더 큰 경우(승우가 유령보다 먼저 도착)에만 진입하는 것으로
장애물, 유령을 피해서 목적지까지 도착하는 비용을 측정할 수 있고

이 비용이 측정된다면 성공 (!=0)
측정에 실패한다면(==0) 실패로 출력하면 된다.

단, 이렇게 BFS로 거리를 계산하는 비용은 크므로
불필요한 연산은 넘겨야한다.

유령이 여럿 있을 수 있음을 전제로 하므로
계산 범주에 모든 유령을 넣는 대신
목적지에 가장 가까운 유령 하나만 포함해서 BFS 하면 된다.

유령마다 속도가 다른게 아니라면
결국 도착여부를 결정하는건 목적지에 가장 가까운 유령이 된다.
```
