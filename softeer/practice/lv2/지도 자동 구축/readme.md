## 지도 자동 구축
> https://softeer.ai/practice/6280

## 평가
### 2024-04-23
```
규칙을 분석해서 공식만 짜면 쉽게 풀 수 있는 문제 
점의 개수는 9, 25, 81.. 로 증가하며
여기서 3**2, 5**2, 9**2 라는 규칙을 유추할 수 있고
3,5,9는 2+1, 4+1, 8+1 이므로 a(n)=(2**n)+1이다.
최종적으로 a(n)=((2**n)+1)**2 임을 도출 할 수 있다.

다른 방법으로는
한변의 점 개수는 3, 5, 9...로
점화식으로 표현하면 a(n+1)=a(n)*2-1 이고
일반항으로 바꾸기 위해
a(n+1)=p*a(n)+q 형식의 점화식에 대입하면
p=2, q=-1 이다.
이는 a(n)=(a(1)+k)*p**(n-1) 로 일반화 되는데
a(1)=3, k=q=-1, p=2 이므로 대입 시
a(n)=(3-1)*(2**(n-1))-(-1)
    =(2**n)+1
이다.
이를 제곱해야하니 최종적으로
a(n)=((2**n)+1)**2
로 일반화 할 수 있다.
```
